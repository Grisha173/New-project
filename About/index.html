Наше производство располагает развитой технической базой на основе оборудования ведущих мировых производителей, позволяющего изготовить технологическую оснастку, как собственной разработки, так и по чертежам заказчика. Производственный цикл по изготовлению пресс-форм и другой технологической оснастки производится на современных станках с ЧПУ. Инструментальное производство осуществляет изготовление следующей технологической оснастки:
  • Штампы холодной листовой штамповки:
    • вырубные;
    • формовочные;
    • совмещенного действия;
    • последовательного действия.
  • Формы:
    • пресс-формы для литья термопластичных материалов, в том числе и горячеканальные;
    • пресс-формы для прямого прессования;
    • вакуум-формы;
    • литейные формы для литья.
  • Приспособления:
    • для сварки;
    • для механической обработки;
    • станочные приспособления;
    • для разметки;
    • приспособления специального назначения.
  • Также инструментальное производство производит следующие виды обработки:
    • фрезерную обработку деталей на универсальных станках и обрабатывающих центрах;
    • шлифовальные работы;
    • электроэрозионную обработку;
    • заточку режущего инструмента;
    • порезку заготовок на ленточно-пильных станках;
    • термообработку конструкционных и инструментальных сталей.
